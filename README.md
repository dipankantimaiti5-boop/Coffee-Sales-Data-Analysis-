# Coffee-Sales-Data-Analysis-
This is a Coffee Sales Exploratory Data Analysis project using Python , Numpy , Pandas, Matplotlib , focusing on Data Cleaning, Data Manipulation.
I have used VS Code editor along with jupyter extension and python 3.14.2 kernels.
I have installed pandas , numpy, seaborn , matplotlib using pip install command on VS code Terminal 
In this analysis i have created pie plot and bar plot along with i have shown the data frame with dimentions 
i have checked null values and shown distribution plot 
i have checked duplicates of the data frame 
i have shown total number of rows and columns 
Under each plot i have added conclusions
